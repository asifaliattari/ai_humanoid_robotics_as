## 3. üí¨ AI Chat Assistant

Click the floating chat button (bottom right) to ask questions about the book.

### Two Modes

#### Mode 1: Search Entire Book
Just type your question:
- "What is ROS 2?"
- "How do I set up Isaac Sim?"
- "What's the difference between Gazebo and Unity?"

The AI searches the entire textbook and provides answers with sources.

#### Mode 2: Ask About Selected Text
1. Highlight any text on the page
2. The chat widget shows "Asking about selected text"
3. Type your question about that specific section

**Example:**
- Select: "Quality of Service (QoS) policies"
- Ask: "Can you explain this in simpler terms?"
- Get: Detailed explanation of the selected concept

### Tips for Best Results

‚úÖ **Do:**
- Be specific: "How do I install ROS 2 on Ubuntu 22.04?"
- Reference sections: "In Module 3, what is cuOpt?"
- Ask follow-ups: "Can you explain that differently?"

‚ùå **Don't:**
- Ask about unrelated topics (AI only knows this textbook)
- Expect it to write full programs (use code snippets as starting points)

## Privacy & Data

- **Translations**: Cached for faster loading (anonymous)
- **Personalization**: Requires account, stored securely
- **Chat History**: Optional logging for improving responses
- **Selected Text**: Never logged or stored

## Troubleshooting

### Translation Failed
- Check internet connection
- Try refreshing the page
- Some special characters may need manual review

### Personalization Not Working
- Make sure you're signed in
- Verify your profile is complete
- Check browser console for errors

### Chat Not Responding
- Verify backend API is running
- Check browser console for errors
- Try refreshing and asking again

## Technical Details

All three features are powered by:

- **Backend**: FastAPI (Python)
- **Vector DB**: Qdrant (semantic search)
- **Database**: Neon Postgres (user data)
- **LLM**: OpenAI GPT-4 Turbo
- **Embeddings**: text-embedding-3-small

For developers interested in how this works, see [AI Features Documentation](/docs/ai-features/).
