# اے آئی مقامی خصوصیات

## جائزہ

یہ کتاب **اے آئی مقامی** ہے: یہ اے آئی کو نہ صرف مطالعے کے موضوع کے طور پر استعمال کرتی ہے، بلکہ سیکھنے کے آلے کے طور پر بھی۔ پلیٹ فارم میں شامل ہے:

1. **RAG چیٹ بوٹ** - کسی بھی موضوع کے بارے میں سوالات پوچھیں
2. **صارف کی تصدیق** - JWT کے ساتھ محفوظ لاگ ان

---

## 1. RAG چیٹ بوٹ

**مقصد**: پوری کتاب کو سیاق و سباق کے طور پر استعمال کرتے ہوئے سوالات کے جوابات دیں

**آرکیٹیکچر**:
- **فرنٹ اینڈ**: React چیٹ ویجیٹ (نیچے دائیں فلوٹنگ بٹن)
- **بیک اینڈ**: FastAPI + Qdrant (ویکٹر سرچ) + OpenAI GPT-4
- **ایمبیڈنگز**: کتاب کا مواد سیکشن کے لحاظ سے چنکڈ، میٹا ڈیٹا کے ساتھ Qdrant میں محفوظ

### چیٹ بوٹ موڈز

#### موڈ A: پوری کتاب سے پوچھیں
**استعمال کا معاملہ**: عام سوالات جیسے "ROS 2 ایکشنز کیسے کام کرتے ہیں؟"

**پائپ لائن**:
```
صارف کا سوال → سوال ایمبیڈ کریں → Qdrant تلاش کریں → ٹاپ-k چنکس حاصل کریں → LLM ترکیب → جواب
```

#### موڈ B: منتخب متن کے بارے میں پوچھیں
**استعمال کا معاملہ**: پیراگراف ہائی لائٹ کریں اور پوچھیں "اسے آسان الفاظ میں سمجھائیں"

**پائپ لائن**:
```
منتخب متن + صارف کا سوال → LLM (کوئی ویکٹر سرچ نہیں) → آسان جواب
```

### مثال تعاملات

**سوال**: "ROS 2 ٹاپکس اور سروسز میں کیا فرق ہے؟"
**جواب**: "ٹاپکس ایک سے زیادہ غیر مطابقت پذیر میسجنگ کے لیے pub/sub استعمال کرتے ہیں (مثلاً، کیمرے کی تصاویر براڈکاسٹ کرنا)، جبکہ سروسز ایک سے ایک مطابقت پذیر استفسارات کے لیے request/response استعمال کرتے ہیں (مثلاً، پلانر سے راستہ مانگنا)۔ دیکھیں [ماڈیول 1: ROS 2](/docs/modules/ros2/)۔"

## 2. صارف کی تصدیق

**مقصد**: محفوظ صارف لاگ ان اور رجسٹریشن

**خصوصیات**:
- JWT پر مبنی تصدیق
- bcrypt کے ساتھ محفوظ پاس ورڈ ہیشنگ
- نیو بار میں ابتدائیات کے ساتھ صارف اوتار
- مستقل سیشنز

**اینڈ پوائنٹس**:
- `POST /api/auth/register` - نیا اکاؤنٹ بنائیں
- `POST /api/auth/login-json` - لاگ ان کریں اور JWT ٹوکن حاصل کریں
- `GET /api/auth/me` - موجودہ صارف کی معلومات حاصل کریں

---

## نفاذ کی تفصیلات

**بیک اینڈ**: `backend/`
- `api/auth/`: تصدیق کے اینڈ پوائنٹس
- `api/rag/`: چیٹ بوٹ کے اینڈ پوائنٹس
- `models/auth.py`: صارف ماڈل

**فرنٹ اینڈ**: `src/components/`
- `RAGChatWidget.tsx`: فلوٹنگ چیٹ بٹن
- `UserNavbarItem/`: صارف اوتار اور لاگ ان
